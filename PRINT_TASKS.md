# 系统支持的打印和生成任务

## 已实现的任务类型

### 1. 销售合同 (sales_contract)
- **功能**: 生成销售合同文档
- **工具**: `contract_validate`, `sales_contract_generate`
- **示例**: 
  - "打印销售合同"
  - "生成合同 客户 xxx 产品 xxx"
  - "帮我做一份销售合同"
- **特点**: 
  - 支持动态产品提取
  - 自动计算金额（数量 × 规格 × 单价）
  - HTML 表格显示
  - 可编辑数量和单价

## 系统特点

### 1. **动态产品提取**
- 支持任意产品型号，无需预定义
- 自动从数据库获取产品信息
- 支持中文数字和阿拉伯数字

### 2. **智能理解**
- 支持多种自然语言表达
- "X 桶 XXX" 格式
- "XXX 要 X 桶" 格式
- "需要 X 桶 XXX" 格式

### 3. **自动计算**
- 销售合同：数量 × 规格 × 单价
- 自动汇总合计

### 4. **HTML 表格显示**
- 任务摘要中显示 HTML 表格
- 可编辑数量和单价
- 自动重新计算金额

## 扩展性

系统设计支持轻松添加新的打印/生成任务：

1. **创建新的工具类** - 继承 `BaseTool`
2. **注册到工具注册表** - 在 `tool_registry.py` 中添加
3. **注册到意图注册表** - 在 `intent_registry.py` 中添加
4. **配置 LLM 理解** - 添加示例和描述
