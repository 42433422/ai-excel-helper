# TabBar 图标说明

## 需要的图标文件

请在以下位置添加对应的 PNG 图标文件（建议尺寸：81x81 像素）：

### 主 Tab 图标（未选中状态）
- `assets/tab/home.png` - 首页图标
- `assets/tab/category.png` - 分类图标
- `assets/tab/cart.png` - 购物车图标
- `assets/tab/profile.png` - 个人中心图标

### 主 Tab 图标（选中状态）
- `assets/tab/home-active.png` - 首页选中图标
- `assets/tab/category-active.png` - 分类选中图标
- `assets/tab/cart-active.png` - 购物车选中图标
- `assets/tab/profile-active.png` - 个人中心选中图标

## 图标要求

- **尺寸**: 81x81 像素
- **格式**: PNG
- **背景**: 透明
- **颜色**: 
  - 未选中：#999999
  - 选中：#1890ff

## 临时解决方案

如果暂时没有图标，可以：

1. 使用微信开发者工具默认图标
2. 从 iconfont.cn 下载合适的图标
3. 使用设计工具（如 Figma、Sketch）自行设计

## 图标资源推荐

- [iconfont.cn](https://www.iconfont.cn) - 阿里巴巴矢量图标库
- [flaticon.com](https://www.flaticon.com) - 免费图标网站
- [icons8.com](https://icons8.com) - 高质量图标资源
