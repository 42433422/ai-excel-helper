# 微信检测工具包 - 使用说明

## 📋 工具概述

本工具包包含微信自动化相关的核心组件，用于微信窗口检测、消息读取、消息发送等功能。

## 📁 文件说明

### 核心功能文件

1. **wechat_window.py** - 微信窗口检测和管理
   - 检测微信窗口是否可见
   - 获取窗口位置和尺寸
   - 窗口激活和管理

2. **wechat_automator.py** - 微信消息发送自动化
   - 自动发送文本消息
   - 文件发送功能
   - 鼠标点击自动化

3. **wechat_reader_final_v2.py** - OCR消息读取
   - 使用OCR技术识别微信聊天内容
   - 支持文字识别和消息提取
   - 时间戳识别

4. **test_window_detection.py** - 窗口检测测试
   - 测试微信窗口检测功能
   - 调试窗口检测问题

5. **debug_windows.py** - 窗口调试工具
   - 调试微信窗口相关问题
   - 提供详细的窗口信息

## 🚀 使用方法

### 方法一：直接导入使用
```python
# 导入微信窗口检测
from wechat_window import WechatWindow

# 导入消息发送
from wechat_automator import WechatAutomator

# 导入OCR读取
from wechat_reader_final_v2 import WechatReader
```

### 方法二：运行测试程序
```bash
# 测试窗口检测
python test_window_detection.py

# 运行窗口调试
python debug_windows.py
```

## 🔧 核心功能示例

### 窗口检测
```python
from wechat_window import WechatWindow

# 创建窗口检测器
window = WechatWindow()

# 检测微信窗口
if window.find_wechat_window():
    print("找到微信窗口")
    # 获取窗口位置
    x, y, width, height = window.get_window_rect()
    print(f"窗口位置: ({x}, {y}) 尺寸: {width}x{height}")
```

### 消息发送
```python
from wechat_automator import WechatAutomator

# 创建消息发送器
automator = WechatAutomator()

# 发送消息
automator.send_message("你好，这是自动发送的消息")
```

### OCR读取
```python
from wechat_reader_final_v2 import WechatReader

# 创建OCR读取器
reader = WechatReader()

# 读取微信聊天内容
messages = reader.read_chat_messages()
for msg in messages:
    print(f"消息: {msg['text']} - 时间: {msg['timestamp']}")
```

## ⚠️ 注意事项

1. **环境要求**
   - Windows系统
   - Python 3.7+
   - 微信客户端已安装并运行

2. **权限要求**
   - 程序需要访问窗口句柄权限
   - 可能需要管理员权限运行

3. **使用限制**
   - 微信窗口必须可见
   - 确保微信版本兼容
   - 遵守微信使用条款

4. **常见问题**
   - 如果无法检测到窗口，请检查微信是否完全加载
   - OCR功能需要良好的图像清晰度
   - 发送消息前确保聊天窗口已激活

## 🔍 故障排除

### 窗口检测失败
- 确保微信客户端正在运行
- 检查微信窗口是否最小化
- 尝试重新启动程序

### OCR识别不准确
- 调整微信窗口大小
- 确保聊天内容清晰可见
- 检查字体设置

### 消息发送失败
- 确认聊天窗口已激活
- 检查输入框位置
- 验证消息内容格式
