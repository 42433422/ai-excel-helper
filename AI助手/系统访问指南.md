# 🎯 数据库管理系统访问指南

## 🌐 系统访问地址

### 主要页面
- **🏠 系统主页**: http://localhost:8080/
- **📊 数据库管理**: http://localhost:8080/database
- **📋 发货单生成**: http://localhost:8080/shipment

## 🚀 快速开始

### 1. 访问数据库管理
1. 打开浏览器，访问: **http://localhost:8080/database**
2. 系统会自动加载所有客户单位列表

### 2. 查看客户产品
1. 从"选择客户单位"下拉菜单中选择一个客户（如：七彩乐园、蕊芯家私等）
2. 系统会自动加载该客户的所有产品信息
3. 包括产品型号、名称、规格、价格、单位等详细信息

### 3. 搜索产品
1. 在"搜索产品"框中输入关键词
2. 支持按产品名称或型号搜索
3. 实时筛选显示结果

### 4. 管理产品
- **添加产品**: 点击"添加产品"按钮，填写产品信息
- **编辑产品**: 点击产品行的编辑按钮（✏️）
- **删除产品**: 点击产品行的删除按钮（🗑️）

## 🎨 界面功能

### 客户信息展示
当选择客户后，系统会显示：
- 单位名称
- 联系人信息
- 联系电话

### 产品表格功能
- **ID**: 产品唯一标识
- **型号**: 产品型号（如：9803、A1(553)等）
- **产品名称**: 完整的产品名称
- **规格**: 产品规格信息
- **价格**: 产品单价（绿色表示正价，红色表示免费）
- **单位**: 计量单位（个、桶等）
- **状态**: 启用/停用状态

### 统计信息
- 实时显示当前客户的产品数量
- 直观的数据统计卡片

## 🔧 操作指南

### 查看特定客户产品示例
1. 访问 http://localhost:8080/database
2. 在客户选择框中选择 "七彩乐园"
3. 查看该客户的所有产品列表
4. 可以看到 PE白底漆、PU净味面漆稀释剂等产品

### 搜索功能测试
1. 在搜索框输入 "PE"
2. 系统会自动筛选出所有包含"PE"的产品
3. 输入 "9803" 可以找到特定型号的产品

### 添加新产品
1. 点击"添加产品"按钮
2. 填写产品信息：
   - 型号: NEW-001
   - 名称: 新产品
   - 规格: 规格: 10KG
   - 价格: 25.50
   - 单位: 个
3. 点击"添加产品"确认
