📄 PDF打印显示应用窗口演示

💡 说明： 此页面演示如何使用新的show_app参数来控制PDF打印时是否显示应用程序窗口。

👁️ 测试1：显示应用窗口打印（默认方式）

使用 show_app: true - PDF应用正常显示，用户可以看到打印进度

🎯 测试2：隐藏应用窗口打印（备选方式）

使用 show_app: false - PDF应用在后台静默运行，不显示窗口

⚙️ 自定义参数打印

可以自定义PDF文件和显示选项

📋 API调用示例

前端JavaScript代码示例：

// 隐藏应用窗口打印
fetch('/api/print/pdf_labels', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        pdf_path: 'PDF文件/订单26-0200099A_标签.pdf',
        copies: 1,
        show_app: false  // 隐藏应用窗口
    })
})
.then(response => response.json())
.then(data => {
    console.log('打印结果:', data);
});

// 显示应用窗口打印
fetch('/api/print/pdf_labels', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        pdf_path: 'PDF文件/订单26-0200099A_标签.pdf',
        copies: 1,
        show_app: true  // 显示应用窗口
    })
})
.then(response => response.json())
.then(data => {
    console.log('打印结果:', data);
});
🔧 修改说明：
• API端点：/api/print/pdf_labels
• 新增参数：show_app (布尔值，默认为false)
• 参数作用：控制ShellExecute的showCmd参数
• false: showCmd=0 (SW_HIDE) - 隐藏窗口
• true: showCmd=1 (SW_SHOW) - 显示窗口
