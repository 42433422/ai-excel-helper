|------|------|--------|
| **混合意图识别引擎** | 规则 + RASA NLU + BERT 三重保障 | ⭐⭐⭐ |

## 升级路径

### v1.0 → v2.0
```bash
# 安装新依赖
pip install -r requirements.txt

# 启动 Web 服务
python run.py

# 访问 http://localhost:5000
```

### v2.0 → v3.0
```bash
# 更新依赖
pip install -r requirements.txt

# 运行数据库迁移
alembic upgrade head

# 启动服务
python run.py

# 启动 Celery Worker (可选)
celery -A celery_app worker --loglevel=info
```
