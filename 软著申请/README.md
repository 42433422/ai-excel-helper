## 📝 软著申请流程

### 第一步：准备材料

1. **源代码文档** ✅
   - 前后各 30 页，共 60 页
   - 每页不少于 50 行
   - 已去除注释和空行
   - 本目录下的 `完整源代码.txt` 可直接使用

2. **软件文档**（需另行准备）
   - 用户手册 或 设计文档
   - 包含功能说明、界面截图、操作流程
   - 30 页以上

3. **申请表**
   - 在中国版权保护中心网站填写
   - 在线打印并盖章

4. **身份证明**
   - 个人申请：身份证复印件
   - 公司申请：营业执照复印件

### 第二步：提交申请

#### 方式一：现场提交

1. 准备纸质材料
2. 前往中国版权保护中心
3. 现场取号排队
4. 提交材料并缴费

#### 方式二：邮寄提交

1. 准备纸质材料
2. 邮寄至中国版权保护中心
3. 等待受理通知

#### 方式三：在线提交（部分地区）

1. 登录中国版权保护中心官网
2. 注册账号并实名认证
3. 在线填写申请表
4. 上传电子版材料
5. 邮寄纸质材料

### 第三步：审查与领证

1. **受理**（1-3 个工作日）
   - 收到受理通知书
   - 缴纳申请费

2. **审查**（30-60 个工作日）
   - 形式审查
   - 实质审查（如有问题会补正）

3. **领证**（审查通过后）
   - 领取著作权证书
   - 或选择邮寄证书

## ⚠️ 注意事项

### 源代码格式要求

✅ **已满足的要求**：
- [x] 每页不少于 50 行代码
- [x] 已去除注释
- [x] 已去除空行
- [x] 前后端代码分别整理
- [x] 提供文件路径标识

❌ **不要包含的内容**：
- [ ] 代码注释
- [ ] 空行
- [ ] 版权声明（除文件头外）
- [ ] 调试代码
- [ ] 第三方库代码

### 材料装订要求

1. **源代码装订**
   - A4 纸打印
   - 左侧装订
   - 页码连续
   - 封面注明软件名称

2. **文档装订**
   - A4 纸打印
   - 左侧装订
   - 包含目录
   - 页码连续

### 费用标准

| 项目 | 费用（人民币） |
|------|---------------|
| 普通申请 | 300 元 |
| 加急申请 | 根据加急程度 500-2000 元不等 |

### 办理时限

| 类型 | 时限 |
|------|------|
| 普通件 | 60 个工作日 |
| 加急件 | 3-30 个工作日（可选） |

### 常见补正原因

1. **材料不齐全**
   - 缺少申请表、身份证明等
   
2. **源代码格式不符**
   - 每页行数不足 50 行
   - 包含大量注释或空行
   
3. **文档不规范**
   - 缺少功能说明
   - 缺少界面截图
   
4. **软件名称问题**
   - 名称不规范
   - 与已有软件重名
