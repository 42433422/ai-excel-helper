## 四、技术特点

### 4.1 前端特点
1. **组件化开发**：高度模块化的 Vue 组件系统
2. **响应式设计**：适配多种屏幕尺寸
3. **状态管理**：使用 Pinia 进行全局状态管理
4. **类型安全**：全面使用 TypeScript
5. **性能优化**：代码分割、懒加载、GPU 加速动画

### 4.2 后端特点
1. **分层架构**：清晰的分层设计（Routes → Application → Domain → Infrastructure）
2. **领域驱动**：DDD 领域驱动设计理念
3. **RESTful API**：标准化的 RESTful 接口设计
4. **数据库 ORM**：使用 SQLAlchemy 进行数据库操作
5. **意图识别**：混合意图识别系统（规则 + AI）

### 4.3 安全特性
1. **身份验证**：JWT Token 认证
