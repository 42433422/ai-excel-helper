# 回撤按钮重新设计计划

## 目标
将回撤按钮重新设计为**流光边框风格**，使其更符合科技感。

## 设计方案

### 流光边框风格特点
- 细边框（1-2px）
- 悬停时边框有流动动画效果
- 保持圆形或圆角矩形
- 颜色使用青色/蓝色科技风格

### 具体设计参数
- **形状**：圆形，44px x 44px
- **边框**：1px solid rgba(0, 255, 255, 0.5)
- **背景**：半透明 rgba(0, 20, 40, 0.6)
- **文字**：「撤回」
- **流光效果**：使用 CSS @keyframes 创建边框流动动画
- **Hover 效果**：边框颜色增强 + 轻微放大

## 修改文件

### 1. vue-dist/index.html
- 更新按钮内联样式，添加流光边框效果
- 添加 CSS @keyframes 动画定义

### 2. vue-dist/static/css/animations/pro-mode.css（如需要）
- 确保样式与新设计一致

## 实现步骤

1. 在 index.html 中添加流光动画 CSS 样式
2. 更新按钮样式：使用 animation 替代 transition
3. 添加 hover 状态：边框流光效果增强
4. 测试验证效果
