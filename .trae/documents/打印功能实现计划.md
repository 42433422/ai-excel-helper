# 打印功能实现计划

## 需求分析
为发货单系统添加直接连接打印机进行打印的功能。

## 实现步骤

### 1. 后端 - 添加打印API
- **文件**: `app_api.py`
- **功能**: 
  - `/api/printers` - 获取可用打印机列表
  - `/api/print/<filename>` - 打印指定文件
- **技术**: 使用 `win32print` (Windows打印机API)

### 2. 后端 - 打印工具类
- **文件**: `print_utils.py` (新建)
- **功能**:
  - 获取系统打印机列表
  - 打印Excel文件
  - 打印状态反馈

### 3. 前端 - 添加打印功能
- **文件**: `index.html`
- **功能**:
  - 添加"选择打印机"下拉框
  - 添加"打印发货单"按钮
  - 显示可用打印机列表

### 4. 集成测试
- 测试打印机选择
- 测试打印功能
- 错误处理和状态反馈

## 技术细节
- 使用 `pywin32` 库的 `win32print` 模块
- 支持Excel文件直接打印
- 提供打印进度和状态反馈
