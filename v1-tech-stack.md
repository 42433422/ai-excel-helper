# XCAGI v1.0 技术栈

> AI Excel Helper 1.0 - 出货单基础自动化

---

## 技术架构

```
┌─────────────────────────────────────┐
│          Excel 出货单                │
└─────────────────────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│     Python 解析器 (openpyxl)         │
└─────────────────────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│         SQLite 数据库                │
└─────────────────────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│      TSC 打印机 (标签打印)            │

---

## 技术明细

| 技术 | 说明 |
|------|------|
| **Python 3.11+** | 核心编程语言 |
| **openpyxl** | Excel 文件读写 |
| **SQLite** | 轻量级本地数据库 |
| **TSC 打印机** | 标签打印输出 |

---

## 核心功能

- ✅ Excel 出货单解析
- ✅ 产品型号提取
- ✅ 数量信息提取
- ✅ 价格信息提取
- ✅ 标签模板生成
- ✅ TSC 打印机集成
- ✅ 批量打印功能

---

## 项目结构

```
AI助手/

- Windows 10/11
- Python 3.11+
- TSC 打印机 (可选)

- 简单的 Excel 出货单处理
- 基础的标签打印需求
- 单机环境使用
