## 项目结构 (DDD 架构)

```
app/
├── domain/              # 领域实体 & 值对象
├── application/         # 应用服务 & 端口接口
├── infrastructure/      # 基础设施实现
│   ├── repositories/    # 仓储实现
│   ├── persistence/     # 持久化
│   ├── lookups/         # 查询服务
│   ├── documents/       # 文档生成
│   └── templates/       # 模板管理
├── routes/             # API 路由
├── services/           # 业务服务
│   └── skills/         # 技能模块
└── utils/             # 工具函数
```
