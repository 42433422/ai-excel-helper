## 项目结构

```
ai-excel-helper/
├── app/                 # 主应用目录
│   ├── routes/          # API 路由
│   ├── services/        # 业务逻辑
│   ├── utils/           # 工具函数
│   ├── db/              # 数据库模型
│   └── config.py        # 配置文件
├── frontend/            # 前端代码
│   ├── src/             # Vue 源码
│   └── public/          # 静态资源
├── templates/           # HTML 模板
├── temp_excel/          # 临时 Excel 文件
├── tests/               # 测试代码
├── requirements.txt     # 依赖文件
└── run.py               # 启动脚本
```
