## 请求处理流程

### 基本对话流程

```
用户
 │
 │ 1. "你好，请介绍一下你自己"
 ↓
┌────────────────────────────────────┐
│        OpenClaw Gateway            │
│  - 接收用户消息                     │
│  - 识别意图                         │
│  - 路由到 DeepSeek 技能              │
└─────────────┬──────────────────────┘
              │
              │ 2. HTTP POST /chat
              │    {"message": "你好...", "use_tools": false}
              ↓
┌────────────────────────────────────┐
│     DeepSeek MCP Server            │
│  - 接收请求                         │
│  - 验证参数                         │
│  - 构建消息历史                     │
└─────────────┬──────────────────────┘
              │
              │ 3. POST /chat/completions
              │    {"model": "deepseek-chat", "messages": [...]}
              ↓
┌────────────────────────────────────┐
│        DeepSeek API                │
│  - 处理消息                         │
│  - 生成回复                         │
└─────────────┬──────────────────────┘
              │
              │ 4. 返回回复
              │    {"choices": [{"message": {"content": "你好！我是..."}}]}
              ↓
┌────────────────────────────────────┐
│     DeepSeek MCP Server            │
│  - 解析响应                         │
│  - 提取回复内容                     │
│  - 格式化响应                       │
└─────────────┬──────────────────────┘
              │
              │ 5. {"reply": "你好！我是...", "usage": {...}}
              ↓
┌────────────────────────────────────┐
│        OpenClaw Gateway            │
│  - 接收响应                         │
│  - 提取回复                         │
│  - 发送给用户                       │
└─────────────┬──────────────────────┘
              │
              │ 6. "你好！我是你的智能助手..."
              ↓
            用户
```

### 带工具调用的流程

```
用户
 │
 │ 1. "帮我查询一下有哪些客户"
 ↓
┌────────────────────────────────────┐
│        OpenClaw Gateway            │
└─────────────┬──────────────────────┘
              │
              │ 2. HTTP POST /chat
              │    {"message": "帮我查询...", "use_tools": true}
              ↓
┌────────────────────────────────────┐
│     DeepSeek MCP Server            │
│  - 构建请求                         │
│  - 添加工具定义                     │
└─────────────┬──────────────────────┘
              │
              │ 3. POST /chat/completions (带 tools)
              ↓
┌────────────────────────────────────┐
│        DeepSeek API                │
│  - 分析意图                         │
│  - 决定调用 get_customers 工具       │
│  - 返回 tool_calls                  │
└─────────────┬──────────────────────┘
              │
              │ 4. {"tool_calls": [{"function": "get_customers"}]}
              ↓
┌────────────────────────────────────┐
│     DeepSeek MCP Server            │
│  - 识别工具调用                     │
│  - 执行 get_customers()             │
│  - 获取客户数据                     │
└─────────────┬──────────────────────┘
              │
              │ 5. 执行工具
              │    返回：{"customers": [...]}
              ↓
┌────────────────────────────────────┐
│     DeepSeek MCP Server            │
│  - 将工具结果添加到消息历史          │
│  - 再次调用 DeepSeek API             │
└─────────────┬──────────────────────┘
              │
              │ 6. POST /chat/completions (带 tool result)
              ↓
┌────────────────────────────────────┐
│        DeepSeek API                │
│  - 根据工具结果生成回复             │
│  - "目前有以下客户：..."            │
└─────────────┬──────────────────────┘
              │
              │ 7. 返回最终回复
              ↓
┌────────────────────────────────────┐
│     DeepSeek MCP Server            │
│  - 解析响应                         │
│  - 返回给 OpenClaw                  │
└─────────────┬──────────────────────┘
              │
              │ 8. {"reply": "目前有以下客户：..."}
              ↓
┌────────────────────────────────────┐
│        OpenClaw Gateway            │
└─────────────┬──────────────────────┘
              │
              │ 9. "目前有以下客户：..."
              ↓
            用户
```

## 错误处理流程

```
请求
 │
 ↓
┌──────────────────────────┐
│   参数验证                │
└─────┬────────────────────┘
      │
      ├─ 参数缺失 ──▶ 返回 400 错误
      │
      ↓
┌──────────────────────────┐
│   调用 DeepSeek API       │
└─────┬────────────────────┘
      │
      ├─ 超时 ──▶ 返回 504 错误
      │
      ├─ 网络错误 ──▶ 返回 503 错误
      │
      ├─ API 错误 ──▶ 返回 500 错误
      │
      ↓
┌──────────────────────────┐
│   执行工具                │
└─────┬────────────────────┘
      │
      ├─ 工具不存在 ──▶ 返回 404 错误
      │
      ├─ 参数错误 ──▶ 返回 400 错误
      │
      ├─ 执行异常 ──▶ 捕获异常，返回错误信息
      │
      ↓
┌──────────────────────────┐
│   成功响应                │
└──────────────────────────┘
```
