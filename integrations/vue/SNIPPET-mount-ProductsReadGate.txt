产品页仍无遮罩时，请逐项确认：

1) 推荐在 App.vue 根只挂一次全局弹窗（任意页都能输入口令）：

   import GlobalReadTokenPrompt from "@/components/fhd/GlobalReadTokenPrompt.vue";

   <template>
     <GlobalReadTokenPrompt :api-base="yourApiOriginOrEmpty" />
     <router-view />
   </template>

   各业务页在 onMounted 监听 window 事件 fhd-db-read-unlocked 后重拉受保护接口（见 dbTokenHeaders.ts 中常量名）。

   若只想产品页弹窗，可改用 ProductsReadGate 挂在 #view-products 内（二选一即可，避免重复弹两次）。

   yourApiOriginOrEmpty：与请求 /api 时用的同源前缀一致；若前端 dev 端口与后端不同，传完整 origin，如 "http://127.0.0.1:8000"。

2) 所有受保护 GET（/api/products/*、/api/customers*、/api/purchase_units、出货单位 等）须带读头。
   推荐在全局 axios 上调用（从 FHD 拷贝 attachAxiosDbReadInterceptor.ts + dbTokenHeaders.ts）：
   import { attachAxiosDbReadInterceptor } from "@/…/attachAxiosDbReadInterceptor";
   attachAxiosDbReadInterceptor(api);
   若请求 URL 为完整 http://127.0.0.1:8000/api/...，拦截器须用 combinedRequestUrl 判断（已内置）。

3) 后端已设置环境变量 FHD_DB_READ_TOKEN 并已重启；未设置时列表不会返回 403，本遮罩不会出现（设计如此）。

4) 复制组件：可直接从本仓库 frontend/src/components/fhd/ 拷贝 ProductsReadGate.vue 与 dbTokenHeaders.ts；或沿用 integrations/vue/ 下同名文件。
